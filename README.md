Loop Extractor
==============

Installation
-------------------------
1. Clone repo
2. Install Intel Advisor
3. Install compilers (Intel, etc)
4. Run setup.sh script
    1. Build Local container image
    2. Download pin
    3. Build pin tool
    4. Build loop extractor
